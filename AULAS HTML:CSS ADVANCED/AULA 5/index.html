Unidades de medida css
  • em -> é o tamanho da fonte declarada. letra M
  • ex -> é o tamanho da fonte declarada. letra X
  • rem -> é o 'em' do root
  • vw -> largura da área visível
  • vw -> altura da área visível
  • vmin -> o menor entre o vw e vh
  • vmax -> o maior entre o vw e o vh
